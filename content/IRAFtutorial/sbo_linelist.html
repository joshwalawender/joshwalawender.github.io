SBO Wavelength Calibration:  He-Ar Linelist


This file is the linelist formatted for IRAF's IDENTIFY command.  Using this list, one will be able to take SBO spectra with a calibration lamp exposure and use the automatic finding algorithm in IDENTIFY to determine a wavelength solution.  To use, download the file, and call IDENTIFY with the option coordlist=sbo_HeAr.dat (you may need to add a path to the sbo_HeAr.dat filename).
